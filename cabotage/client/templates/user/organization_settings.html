Setup Instructions
  1. Configure your tailnet policy in your tailnet policy file. Add tag owners and auto-approvers: 
    "tagOwners": {
  "tag:k8s-operator": [],
  "tag:k8s": ["tag:k8s-operator"]
},
"autoApprovers": {
  "services": {
    "tag:k8s": ["tag:k8s"]
  }
}
  2. Go to Tailscale Admin Console > Trust Credentials.
  3. Click Credential > OAuth.
  4. Enable scopes: Devices Core (Write), Auth Keys (Write), Services (Write).
  5. Assign tag: tag:k8s-operator.
  6. Click Generate credential and paste the client ID and secret below.
